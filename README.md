# TP-JAVA

## Description
Ce projet regroupe plusieurs exercices et mini-jeux réalisés en Java dans le cadre du TP professionnel. Il inclut des implémentations de jeux comme les allumettes, le Fibonacci, les cartes, ainsi que des interfaces graphiques avec Swing.

## Instructions d'installation et d'exécution
1. Cloner le dépôt ou télécharger les fichiers.
2. Compiler les fichiers Java :
	- Utiliser les scripts `compile.bat` et `run.bat` dans le dossier `ProjetExercices` pour compiler et exécuter les exercices principaux.
	- Pour les autres exercices, compiler manuellement les fichiers Java dans chaque dossier.
3. Lancer le menu principal avec `Lancer_Menu.bat` pour accéder aux différents jeux et exercices.

## Fonctionnalités principales
- Jeu des allumettes
- Suite de Fibonacci
- Jeu de cartes
- Interfaces graphiques Swing (panels, boutons, radio, etc.)
- Gestion des joueurs et des résultats
